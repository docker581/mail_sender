# test_task_mailganer

## Описание проекта
Cервис отправки email рассылок

## Стек технологий
- Python 2.7.18
- Django 1.11.29
- Celery 4.4.7

## Команды
### Клонирование репозитория
```bash
git clone https://github.com/docker581/test_task_mailganer
```

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск приложения
```bash
python manage.py runserver
```
